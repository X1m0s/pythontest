# Python 字符串格式化

## 格式化符号

| 符号 | 描述 |
|------|------|
| `%c` | 格式化字符（按 ASCII 码） |
| `%s` | 格式化字符串 |
| `%d` | 格式化十进制整数 |
| `%u` | 格式化无符号整数（已废弃，同 `%d`） |
| `%o` | 格式化无符号八进制数 |
| `%x` | 格式化无符号十六进制数（小写） |
| `%X` | 格式化无符号十六进制数（大写） |
| `%f` | 格式化浮点数，可指定精度 |
| `%e` | 科学计数法格式化（小写 e） |
| `%E` | 科学计数法格式化（大写 E） |
| `%g` | `%f` 和 `%e` 的简写（自动选短的） |
| `%G` | `%f` 和 `%E` 的简写 |
| `%p` | 十六进制格式化变量地址（已废弃） |

## 辅助指令

| 符号 | 功能 | 示例 | 结果 |
|------|------|------|------|
| `*` | 动态定义宽度/精度 | `"%*d" % (5, 10)` | `   10` |
| `-` | 左对齐 | `"%-5d" % 10` | `10   ` |
| `+` | 正数前显示 `+` | `"%+d" % 10` | `+10` |
| ` ` | 正数前显示空格 | `"% d" % 10` | ` 10` |
| `#` | 显示进制前缀 | `"%#o" % 8` | `0o10` |
| `0` | 左侧用 `0` 填充 | `"%05d" % 10` | `00010` |
| `%%` | 输出字面 `%` | `"50%%"` | `50%` |
| `(key)` | 字典映射取值 | `"%(n)s" % {"n":"A"}` | `A` |
| `m.n` | 最小宽度 m，精度 n | `"%8.2f" % 3.14` | `    3.14` |

## 代码示例

```python
# 基础
"Hello %s, age %d" % ("Tom", 25)       # Hello Tom, age 25

# 精度
"%.2f" % 3.14159                        # 3.14

# 宽度 + 对齐
"%-10s" % "hi"                          # hi        

# 正数显示加号
"%+d" % 10                              # +10

# 进制
"%o" % 8                                # 10
"%x" % 255                              # ff
"%#x" % 255                             # 0xff

# 字典映射
data = {"name": "Alice", "score": 95}
"%(name)s: %(score)d" % data            # Alice: 95

# 动态宽度
"%*.*f" % (8, 2, 3.14)                  #     3.14

# 左侧填充0
"%05d" % 10                             # 00010

# 字面%
"50%%"                                  # 50%
